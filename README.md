### 💼 Sobre mim

Desenvolvedor Full Stack com foco em construir aplicações web e mobile escaláveis utilizando **Node.js**, **React**, **Next.js** e **TypeScript**. Tenho experiência em construção de APIs REST, modelagem de bancos de dados relacionais com PostgreSQL e integração com serviços **AWS** (Lambda, API Gateway, S3, DynamoDB, Cognito, SQS).

Atualmente atuo como freelancer desenvolvendo CRMs, landing pages e sistemas B2B, com vivência em metodologias ágeis e boas práticas de versionamento. Também exploro integrações com **IA generativa** (Google Gemini, Vertex AI) e arquiteturas **RAG** em projetos próprios.
