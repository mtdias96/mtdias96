# Olá, me chamo Matheus Dias! 👨‍💻

Tenho 29 anos e sou **desenvolvedor full stack** apaixonado por tecnologia, inovação e resolução de problemas. Minha jornada no mundo da programação começou aos 27 anos, e desde então venho buscando aprender e crescer cada vez mais. Ao longo dos anos, desenvolvi habilidades em desenvolvimento de software, arquitetura de sistemas, deploy em servidores e gestão de projetos.

Com experiência prática em desenvolvimento de aplicações web e mobile, tenho focado principalmente em tecnologias como **React.js, React Native, Node.js, Next.js, TypeScript, PostgreSQL**, além de explorar infraestrutura de nuvem com **AWS** e **Serverless Framework**. Acredito que o aprendizado contínuo é a chave para o sucesso, e é por isso que estou sempre me desafiando a experimentar novas tecnologias e ferramentas para melhorar minha performance e criar soluções ainda mais eficientes.

## 🚀 Minhas Especialidades

- **Desenvolvimento Full Stack:** Criação de aplicações completas usando React.js, Node.js, Next.js e serviços da AWS.  
- **Arquitetura de Software:** Construção de sistemas escaláveis e robustos seguindo princípios de boas práticas como Clean Architecture e DDD.  
- **Desenvolvimento Mobile:** Criação de aplicativos móveis com React Native.  
- **Backend Serverless:** Implementação de APIs serverless com AWS Lambda, API Gateway e integração com outros serviços da AWS.
